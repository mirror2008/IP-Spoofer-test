# IP Spoofer Test（IP伪造检测脚本）

基于 CAIDA 的 Spoofer 工具封装的自动化脚本，用于检测当前服务器是否允许 **IP 源地址伪造（IP Spoofing）**。

👉 适用于：
- VPS / 云服务器安全检测
- 网络环境合规性检测（BCP38）
- 安全研究 / 渗透测试环境分析

---

## 🚀 项目地址

脚本文件：

👉 https://github.com/mirror2008/IP-Spoofer-test/blob/main/spoofer_auto.sh

---

## 📦 功能特点

- 自动下载 Spoofer 源码
- 自动安装依赖
- 自动编译（支持 Debian / Ubuntu）
- 自动运行测试（无需手动输入）
- 自动解析结果
- 输出清晰表格总结

---

## ⚙️ 使用方法

### 1️⃣ 克隆项目

```bash
git clone https://github.com/mirror2008/IP-Spoofer-test.git
cd IP-Spoofer-test
```

---

### 2️⃣ 赋予执行权限

```bash
chmod +x spoofer_auto.sh
```

---

### 3️⃣ 运行脚本（必须 root）

```bash
sudo ./spoofer_auto.sh
```

---

## 🧠 结果说明

| 状态 | 含义 |
|------|------|
| ❌ 不可伪造 | 网络已启用源地址校验（安全） |
| ⚠️ 可伪造 | 存在 IP 伪造能力（部分风险） |
| 🔴 高危 | 可伪造公网 IP（严重风险） |

---

## ⚠️ 注意事项

- 需要 root 权限（发送原始数据包）
- 测试过程会发送 UDP 探测包
- 建议在 VPS / 测试环境使用
- 不建议在生产高敏感环境频繁运行

---

## 🔍 原理说明

该脚本调用 CAIDA 的 Spoofer：

- 构造伪造源地址的数据包
- 发送到远端测试服务器
- 判断是否成功发送 / 接收
- 分析网络是否部署源地址校验（BCP38）
